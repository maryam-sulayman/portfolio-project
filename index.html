World Clock

For my world clock project, I was asked to create an app showing the current time across various cities around the world. During this 2-week workshop, I learned about HTML inputs and was taught to use moment.tz() to pull out the current time in different timezones.
This workshop was great and I was able to create a clock app which displays the current time for a list of various cities. This time is also being updated every second which I find very cool. The user is also able to select a city from the drop-down menu at the top of the app to find out the current time in those timezones. The user is also able to find out their current time by clicking on the location icon next to the select menu. This was built using moment.tz.guess()
